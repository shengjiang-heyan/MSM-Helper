# MSM Helper

## 介绍
这是一个自动化脚本，用于执行游戏《My Singing Monsters》(萌怪合唱团) 中的重复性操作。

## 功能
- 自动采集金币（目前仅为点击一键采集）、钻石、食物。

## 运行方式
下载Releases中的发行文件，双击运行。
脚本采用图像识别运行，因此不容易出现误触的情况。

**注意：若脚本执行情况异常，请将鼠标拖动到屏幕最左上角并保持一秒，程序将自动崩溃并退出。**

当脚本运行结束后，在exe文件的同目录下会生成gameLog.txt，其中会记载日志信息。
